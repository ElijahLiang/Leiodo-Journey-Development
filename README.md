## Local Development

1. Install [Hugo Extended](https://gohugo.io/installation/) (v0.146.0+)
2. Clone this repository with submodules:
   ```bash
   git clone --recurse-submodules https://github.com/ElijahLiang/blog.git
   ```
3. Start the development server:
   ```bash
   hugo server -D
   ```
4. Open http://localhost:1313/blog/ in your browser

## Writing New Posts

Create a new post:
```bash
hugo new posts/your-post-title.md
```

## Deployment

This blog is automatically deployed to GitHub Pages via GitHub Actions when you push to the `main` branch.
